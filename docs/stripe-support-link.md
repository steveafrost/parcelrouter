ParcelRouter uses a Stripe-hosted Payment Link for optional one-time support contributions. This keeps card handling on Stripe and avoids custom payment code in the static marketing site.

## Notes

This is intentionally a fixed-price "buy us a coffee" link. Stripe Payment Links also support pay-what-you-want donations from the Stripe Dashboard, but this repository currently uses the fixed-price link created through the connected Stripe tool.
